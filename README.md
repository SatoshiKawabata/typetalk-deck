# TypetalkDeck

[Typetalk](https://www.typetalk.com) client app like [TweetDeck](https://tweetdeck.twitter.com/).

## setup
### Add `secret.json` file.
```json
{
  "client_id": "OUR_CLIENT_ID",
  "client_secret": "OUR_CLIENT_SECRET"
}
```

### npm install
```
npm i
```

## npm scripts
### start web view app
```
npm start
```

#### build web view app
```
npm run build
```

### start electron
```
npm run electron
```

### fix tslint
```
npm run tslint
```

### test
```
npm run test-watch
```

#### test
```
npm run test
```
