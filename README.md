# TypetalkDeck

## setup
### Add `secret.json` file.
```json
{
  "client_id": "YOUR CLIENT ID",
  "client_secret": "YOUR SECRET"
}
```

### npm install
```
npm i
```

## npm scripts
### start web view app
```
npm start
```

#### build web view app
```
npm run build
```

### start electron
```
npm run electron
```

### fix tslint
```
npm run tslint
```

### test
```
npm run test-watch
```

#### test
```
npm run test
```
